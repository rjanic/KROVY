# ACAD KROVY – roadmapa

## 0.1.0 – inteligentné čiary a výkaz

- [x] Dátový model prvkov krovu.
- [x] Uloženie dát pri entite DWG (`Extension Dictionary` + `Xrecord`).
- [x] Hromadné priradenie a hromadná úprava prierezu.
- [x] Základná dĺžka z 2D čiary/polyline.
- [x] Prepočet krokvy/vzpery podľa jedného sklonu.
- [x] Výkaz z výberu a z celého modelového priestoru.
- [x] Tabuľka v AutoCADe.

## 0.2.0 – pracovný komfort

- [ ] Ribbon karta **ACAD KROVY** a PNG/SVG ikonky.
- [ ] Kontextové menu pravým tlačidlom.
- [ ] Menovky K1, K2, P1 pri prvkoch a ich aktualizácia.
- [ ] Príkazy „Vybrať všetky krokvy“, „Vybrať všetky väznice“, „Vybrať podobné“.
- [ ] Filtrovanie prvkov podľa typu, prierezu, materiálu a strešnej roviny.
- [ ] Duplicita/výstraha pri neoznačenom prvku.

## 0.3.0 – strešné roviny

- [ ] Objekt strešnej roviny v 2D: hrebeň, odkvap, sklon, smer spádu.
- [ ] Automatické priradenie prvku k rovine.
- [ ] Skutočná priestorová dĺžka pre ľubovoľný smer v rovine.
- [ ] Podpora nárožných a úžľabných krokiev.
- [ ] Kontrola, či prvok leží v zvolenej strešnej rovine.

## 0.4.0 – výkaz pre výrobu

- [ ] Detailný zoznam všetkých kusov s označením.
- [ ] Súhrnný rezný plán podľa obchodných dĺžok.
- [ ] Nastaviteľné zaokrúhľovanie a prídavky podľa typu prvku.
- [ ] Export Excel, CSV a PDF.
- [ ] Materiálové predvoľby a cenník.

## 1.0.0 – predajná verzia

- [ ] Inštalátor a automatické načítanie `.bundle`.
- [ ] Aktivácia licencie, aktualizácie a diagnostika.
- [ ] SK/CZ/EN používateľské rozhranie.
- [ ] Dokumentácia a ukážkové DWG výkresy.

## Mimo rozsahu prvej verzie

Statické posúdenie a odporúčanie prierezov nie sú súčasťou 0.1.0. Taký modul by musel riešiť normové zaťaženia, snehovú oblasť, vietor, rozpätie, triedu dreva a právnu zodpovednosť.
